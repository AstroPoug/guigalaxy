--- title: "Windows 95" date: 1995-08-24 description: "Released in August 1995, Windows 95 brought many changes to the Windows operating system. It introduced the start menu, taskbar, greater usage of right-click menus, a desktop that can have icons put on it, preemptive multitasking, and so on, as well as being the first consumer-based version of Windows that could truly be called a real operating system instead of an operating environment that run under DOS (despite still having DOS underpinnings at its core). Its impact was so immense, it even went on to influence other operating systems for years to come." ---
